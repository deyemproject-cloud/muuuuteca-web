# MuuuTeca — Il Regno della Carne

Website for MuuuTeca, an Italian steakhouse in Pavia, Italy.

## Overview

- **Type:** Restaurant / Steakhouse website
- **Location:** Via Porta Damiani 30, Pavia
- **Services:** Premium beef, curated wine cellar, grill dining
- **Design:** Dark luxury aesthetic — gold (#b8860b) accents on deep black
- **Fonts:** Montserrat (Google Fonts)

## Stack

- Single-file HTML (no build step required)
- Tailwind CSS via CDN
- Google Fonts
- Material Symbols Icons
- Vanilla JS (no frameworks)

## Features

- Fully responsive (mobile-first)
- Smooth scroll navigation
- Fixed floating nav with glass blur effect
- Mobile hamburger drawer menu
- Active nav link highlighting on scroll
- Hero parallax effect
- Image hover animations
- Booking form with validation
- Italian language throughout

## Sections

1. **Hero** — Full-bleed atmospheric image with headline + CTAs
2. **La Nostra Missione** — 3-column mission cards + narrative block
3. **Dove la Carne incontra il Vino** — Wine cards grid
4. **Prenotazioni** — Booking form with date/time/guests
5. **Footer** — Brand, links, copyright

## Deploy

Static site — host on GitHub Pages, Netlify, Vercel, or any CDN.

```bash
# Local preview
open index.html
```
